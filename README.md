# Todo List Web App

This is a simple To-Do List web Application build using **Python Flask** and **SQLlite**.

It allows users to:
-View their tasks
- Add new tasks
- Update Tas
- Dalete Task

## Technologies Used
- Python
- Flask(Web Framework)
- SQLite(Database)
- BootStrap (for Styling)
